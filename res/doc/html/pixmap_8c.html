◆ imc_pixmap_to_ascii()

ImcError_t imc_pixmap_to_ascii ( Pixmap_t * pixmap,
const char *const fname )

Outputs pixmap image as ASCII art to the file specified by "fname".

In order to use this function, it is advised that you first make your font as small as possible. To get the number of columns/rows of your terminal, use tput lines and tput cols. Run cat <fname> to print the contents to the TTY.

Warning
Results will vary depending upon whether you're using RGB or RGBA. This is because RGB uses the color channels to determine luma, whereas, RGBA uses the alpha channel. RGBA cannot use the color channels since that would make this function incompatible with imc_pixmap_to_grayscale() or imc_pixmap_to_monochrome(), which strips the color data.
Since
21-07-2024
Parameters
pixmapA Pixmap_t containing the image that shall be converted into ASCII art
fnameThe name of the file that the ASCII art shall be output to
Returns
An ImcError_t representing the exit status code
